# Audio Genre Classifier 🎵

A machine learning project to classify music genres using signal processing and deep learning.

## Dataset
GTZAN Dataset — 1000 audio clips across 10 genres (blues, classical, country, disco, hiphop, jazz, metal, pop, reggae, rock)

## Approach
- Exploratory Data Analysis — waveforms and spectrograms
- Feature extraction using librosa (MFCCs, spectral features)
- Classical ML models (Random Forest, SVM)
- CNN on Mel spectrograms

## Tech Stack
Python, librosa, scikit-learn, PyTorch, matplotlib
